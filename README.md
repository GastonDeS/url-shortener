
# URL-Shortener

## Requerimentos

Para correr el proyecto es necesario ejecutar los siguientes comandos:  
Desde la carpeta root
```bash
  cd frontend  
  npm install
  ```

```bash
  cd nodeBackend  
  npm install
```

## Deployment

Desde la carpeta root
```bash
  cd nodeBackend  
  npm run dev
```

```bash
  cd frontend  
  npm run start
  ```
